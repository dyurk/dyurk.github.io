ERROR 2006 (HY000) MySQL Server Has Gone Away

The server’s default max_allowed_packet value is 1MB and it looks like it has exceeded the default max. You can increase this if the server needs to handle big queries. This can be done in two ways:

Start MySQL client with the following option (64 MB for this example)

mysql --max_allowed_packet=64M

or Add the following in your my.cnf file

max_allowed_packet=64M
